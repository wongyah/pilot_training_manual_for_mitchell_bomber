# 工程师

![飞机工程师][img_preflight_general]

评估一下即将担任您工程师的这个人。这个人应该比机组其他任何成员都更了解您将要驾驶的飞机。如果他的训练存在不足，您也许可以加以补充。

回想一下您自己的训练。在许多课程中，各种各样的知识从四面八方向您涌来。您必须专注于如何驾驶飞机；在设备方面，您学会了越来越多地依赖士兵，特别是机组组长和工程师的建议。

飞行员和工程师必须密切合作，互相补充和完善彼此教育中的空白。

要成为一名合格的作战工程师，一个人必须彻底了解他的飞机、发动机和武器装备。

他必须与副飞行员密切合作，检查发动机运转、燃油消耗和所有设备的操作。

他必须能够与轰炸员合作，并知道如何 cock、lock 和 load 炸弹挂架。您有责任确保他熟悉这些职责，如果他对这些职责模糊不清，请让轰炸员给予他特别的帮助和指导。

他应该对无线电设备有大致的了解，并能够协助调谐发射机和接收机。

您的工程师应该是您关于飞机信息的主要来源。他应该比机组任何其他成员（包括您在内）都更了解设备。

反过来，您是他关于飞行的信息来源。在与工程师的所有讨论中都要牢记这一点。您能使他对设备每项功能背后的原因有越完整的了解，他在机组中就越有价值。总有一天，他的额外知识可能在紧急情况下拯救全局。

一般而言，在紧急情况下，工程师是您首先求助的人。培养他的自豪感、信心和知识。亲自了解他；检查他知识的广度。让他成为您可以信赖的人。
