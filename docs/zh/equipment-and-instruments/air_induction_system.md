# 空气进气系统

![空气进气系统][img_air_induction_system]

化油器加热控制提供了一个替代的空气源，允许从汽缸头周围流动的热空气进入系统。

通常在空气混合室中有一个挡板。通过冷空气进气口水平进入的空气被偏转向下进入化油器入口。在多尘的大气条件下，空气混合室中的空气滤清器会替代挡板。
